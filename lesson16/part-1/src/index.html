Lerning JS methods

addClass
- добавление класса по клику
removeClass
- удаление класса по клику
toggleClass
- переключение класса по клику
getAttr
- получение какого-то атрибута кнопки по клику
setAttr
- присвоение какого-то атрибута кнопке по клику
alertOnClick
- вызов всплывающего окна по клику
triggerAlert
- тригер alertOnClick кнопки по клику
cloneThis
- по клику склонировать кнопку, дописать текст "(cloned)" и вставить её сразу после этой кнопки
closestElement
- по клику вывести в консоль ближайшего родителя.
eachBtnText
- по клику циклом пройтись по всем кнопкам и собрать массив текста этих кнопок. То есть, в итоге, вывести в консоль массив текста кнопок. Будет это выглядеть примерно так - ["addClass", "removeClass", "toggleClass", ... ]
findMe
- по клику найти эту кнопку в теге body и вывести в консоль результат поиска
fadeInText
- расположить скрытый текст рядом с кнопкой, по клику на кнопку показать этот текст через fadeIn.
fadeOutText
- расположить текст рядом с кнопкой, по клику на кнопку скрыть этот текст через fadeOut.
hideText
- расположить текст рядом с кнопкой, по клику на кнопку скрыть этот текст через hide.
showText
- расположить скрытый текст рядом с кнопкой, по клику на кнопку показать этот текст через show.
dataAboutMe
- по клику вывести следующие данные через запятые в консоль: высота кнопки, ширина кнопки, абсолютную позиция по обеим осям, значение всех атрибутов, родительский элемент, соседние кнопки сверху и снизу, текст кнопки.
